# BlueprintGetter

- **功能描述：** 采用一个自定义的get函数来读取。
  如果没有设置BlueprintSetter或BlueprintReadWrite, 则会默认设置BlueprintReadOnly.

- **使用位置：** UFUNCTION, UPROPERTY

- **引擎模块：** Blueprint

- **元数据类型：** string="abc"
